# Tmou 2019 Qualification Solver
Uses dictionary attack to solve all problems in qualification. It takes around 4 minutes.

## How to use
Python3 is needed. Install requirements and run :-)
```bash
$ pip install -r requirements.txt
$ python main.py
```
